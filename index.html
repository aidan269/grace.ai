How it works

One URL.
Everything else is Grace.

01 —
Research
Grace fetches the article, gathers community intelligence from X/Twitter, and scores the threat on novelty, named actors, and impact before writing a word.
02 —
Generate
Produces a complete cantinasec plugin — a command shim and a full SKILL.md with detection steps, IOCs, risk classification, and remediation.
03 —
Ship
Creates a project folder, initializes a git repo, commits everything, and outputs a one-line push command. Done.
